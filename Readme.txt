                                    

                                       TRAVEL WEBSITE( Project Trang web Du lịch )

     1. Giới thiệu: 
        Website này được xây dựng nhằm quảng bá du lịch cho tỉnh Nghệ an và một số địa điểm du lịch trong nước nổi bật.              
        Trong Website giới thiệu đầy đủ chi tiết các địa điểm du lịch ở các huyện trong tỉnh Nghệ an và đất nước Việt nam!           

     2. Website chạy trên các phiên bản của các trình duyệt, trong các hệ điều hành Linux, Mac, Windows.                                                       

     3. Công nghệ sử dụng:                                                                                                         
         + Trong website chủ trọng công nghệ Bootstrap (4), html và css thuần,                                                       
         + Trang web được thiết kế responsive - thiết kế web đáp ứng trên các giao diện laptop, tablet và mobilphone                 

  
     3.1, Cách cài đặt Bootstrap sử dụng trong dự án, có 2 cách để bắt đầu sử dụng Bootstrap (Trong dự án sử dụng cách 1):       
     3.1.1, Include Bootstrap 4 từ CDN: lên trang chủ getBootstrap.com hoặc vào Bootstrap W3S để lấy link CDN nhúng vào thẻ 
          head các trang web. Cụ thế nhúng vào các CDN sau:                                                                       
          <!-- Latest compiled and minified CSS -->                                                                                
          <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.5.2/css/bootstrap.min.css">                     
                                                                                                                              
          <!-- jQuery library -->                                                                                                 
          <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>                                 
 
          <!-- Popper JS -->                                                                                                       
          <script src="https://cdnjs.cloudflare.com/ajax/libs/popper.js/1.16.0/umd/popper.min.js"></script>                        
                                                                                                                              
          <!-- Latest compiled JavaScript -->                                                                                      
          <script src="https://maxcdn.bootstrapcdn.com/bootstrap/4.5.2/js/bootstrap.min.js"></script>                              
                                                                                                                              
     3.1.2, Download Bootstrap 4 từ trang chủ getbootstrap.com (link: https://getbootstrap.com/docs/5.3/getting-started/download/)                                                                                                        
                                                                                                                              
      + Chú thích các lớp được sử dụng trong dự án:                                                                               
      * .container, .container-fluid: để bắt đầu sử dụng bootstap ta bao bọc 2 lớp này cho các phần tử web                         
      > .container lớp có căn lề  margin,                                                                                   
      > .container-fluid lớp có độ rộng 100%                                                                                
      * lớp .row để cho các phần tử nằm trên một dòng                                                                          
      * class="col-sm-12 col-md-6 mb-2", ý nghĩa: Trên giao diện tablet(sm) các phần tử nằm trên một cột, trên laptop(md)chia thành 
        2 cột, lớp .mb-2 là thuộc tính css: margin-bottom = 0.5rem                                                   
                                                                                                                              
      + Các component bootstrap được sử dụng trong dự án:                                                                         
      * Carousel (link: https://getbootstrap.com/docs/5.3/components/carousel/ ) hoặc ta có thể thao tác nhanh trên Visual Studode Code,
       cú pháp: gõ bs4-car nhấn đồng thời phím Ctrl + Space Bar, dùng các phím mũi tên lên xuống để chọn các mẫu Carousel sau đó nhấn 
       Tab hoặc Enter.                                                                                   
                                                                                                                              
      * Navbar (link https://getbootstrap.com/docs/5.3/components/navbar/) hoặc ta có thể thao tác nhanh trên Visual Studode Code,
        cú pháp: bs4-nav nhấn đồng thời phím Ctrl + Space Bar, dùng các phím mũi tên lên xuống để chọn các mẫu Navbar sau đó nhấn Tab hoặc Enter.                                                                                     

      3.2 Sử dụng CSS, HTML:                                                                                                         
      * Nhúng CSS vào trang HTML, có 3 cách viết css: Inline, Internal, External, Trong dự án này sử dụng External(nhúng vào phần thẻ head)                        * Sử dụng các phần tử giá ::before, ::after                                                                                                                             
      * Sử dụng thuộc tính CSS nâng cao:                                                                        
      > CSS 2D Transforms                                                                                                    
      > CSS 3D Transforms
